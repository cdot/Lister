Items are checked by touching the check box (or by a touch anywhere on the row, if enabled). Check boxes can be placed on either side of the display.

A long touch on an item lets you @action_rename or @action_delete it.

Unless sorting is enabled, a handle on each item lets you move the item to another position in the list. Touch and hold to drag the item.

Actions

@action_pin / @action_unpin
Pin/Unpin the list. When a list is pinned, it will stay in the foreground even when the device is locked.
@action_alpha_sort_on / @action_alpha_sort_off
Toggle alphabetic sorting. This only shows the list as sorted, it doesn't affect the underlying ordering. When this is enabled, you can't manually sort the items
@action_save
This action will appear if the list has to be saved because a previous save failed
@action_item_add_on / @action_item_add_off
While adding is on you can add many items
@action_uncheck_all
@action_check_all
@action_delete_checked
@action_undo_delete
Reverts the last delete, for as long as the list is open. When you exit the list view, the undo history is lost.
@action_rename
@action_share_list
Share this list with any content provider that your phone supports e.g. the SD card, or Gmail, or Google Drive if the Drive app is installed, in a number of different formats. When exporting to JSON, this will create a file with one list in it that can then be imported using the @action_import_lists action on the first screen.
@action_list_settings
Change settings that are specific to this list.
