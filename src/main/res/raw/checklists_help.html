Open a list by touching its name.

A long touch on the list name will bring up a context menu that lets you @action_rename or @action_delete the list, or @action_copy.

@action_copy will make an identical copy of the list. You can use this to create template list with e.g. the basic groceries that you always buy.

Unless alphabetic sorting is on, a handle on each list lets you move the list to another position in the list of lists. Touch and hold to drag the list.

Actions

@action_save
This action will appear if the list has to be saved because a previous save failed.
@action_alpha_sort_on / @action_alpha_sort_off
This only shows the lists as sorted, it doesn't affect the underlying ordering. When sorting is on, you can't manually sort the lists.
@action_new_list
You will be prompted for the list name
@action_import_lists
The lists can come from any content provider that your device supports e.g. the SD card, or Google Drive if the Drive app is installed. Two formats are supported, JSON as previously exported from Lister, or CSV (comma-separated value) format with columns:
List name Item Checked ("F" or "T")
e.g.
Shopping Bread F
Shopping Milk F
Wishes World Peace F
Wishes Maserati F
CSV format can be exported from spreadsheet programs.
@action_preferences
Change settings that are specific to this device.
@app_name is free of charge and free of ads. Source code is available from https://github.com/cdot/Lister
