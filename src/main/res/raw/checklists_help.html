Open a list by touching its name.

A long touch on the list name will bring up a context menu that lets you:

  • Rename List
  • Delete
  • Create Copy

Create copy will make an identical copy of the list. You can use this to create template list with e.g. the basic groceries that you always buy.

Unless alphabetic sorting is on, a handle on each list lets you move the list to another position in the list of lists. Touch and hold to drag the list.

Actions

Save
This action will appear if the list has to be saved because a previous save failed.
Sort on/off
Toggle alphabetic sorting. This only shows the lists as sorted, it doesn't affect the underlying ordering. When this is enabled, you can't manually sort the lists.
New list
Create a new list
Import List
Import lists. The lists can come from any content provider that your device supports e.g. the SD card, or Google Drive if the Drive app is installed. Two formats are supported, JSON as previously exported from Lister, or CSV (comma-separated value) format with columns:
List name Item checked ("F" or "T")
e.g.
Shopping Bread F
Shopping Milk F
Wishes World Peace F
Wishes Maserati F
CSV format can be exported from spreadsheet programs.
Preferences
Change settings that are specific to this device.
