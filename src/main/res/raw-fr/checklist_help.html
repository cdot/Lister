Les éléments sont vérifiés en touchant la case à cocher (ou en appuyant n'importe où sur la ligne, si elle est activée). Les cases à cocher peuvent être placées de chaque côté de l'écran.

Un appui long sur un élément vous permet de @action_rename ou de @action_delete.

À moins que le tri ne soit activé, une poignée sur chaque élément vous permet de déplacer l'élément vers un autre position dans la liste. Appuyez de manière prolongée pour faire glisser l'élément.

Actions

@action_alpha_sort_on / @action_alpha_sort_off
Activez le tri alphabétique. Cela ne montreem> que la liste triée, cela n'affecte pas le commande sous-jacente. Lorsque cette option est activée, vous ne pouvez pas trier manuellement les éléments
@action_save
Cette action apparaîtra si la liste doit être enregistrée car une sauvegarde précédente a échoué
@action_item_add_on / @action_item_add_off
Lorsque l'ajout est activé, vous pouvez ajouter de nombreux éléments.
@action_uncheck_all
@action_check_all
@action_delete_checked
@action_undo_delete
Annule la dernière suppression, tant que la liste est ouverte. Lorsque vous quittez la vue de liste, l'historique des annulations est perdu.
@action_rename
@action_share_list
Partager cette liste vers n'importe quel fournisseur de contenu pris en charge par votre téléphone, par ex. la carte SD, l'e-mail, ou Google Drive si l'application Drive est installée, dans un certain nombre de formats différents. Lors de l'exportation vers JSON, cela créera un fichier avec une liste qui pourra ensuite être importée en utilisant l'option @action_import_lists sur le premier écran.
@action_preferences
Modifiez les paramètres spécifiques à cette liste.

@app_name est gratuit et sans annonces. Le code source est disponible sur https://github.com/cdot/Lister
