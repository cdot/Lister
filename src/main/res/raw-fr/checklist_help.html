Les éléments sont vérifiés en touchant la case à cocher (ou en appuyant n'importe où sur la ligne, si elle est activée). Les cases à cocher peuvent être placées de chaque côté de l'écran.

Un appui long sur un élément vous permet de le renommer ou de le supprimer.

À moins que le tri ne soit activé, une poignée sur chaque élément vous permet de déplacer l'élément vers un autre position dans la liste. Appuyez de manière prolongée pour faire glisser l'élément.

Toggle alphabetic sorting. This only shows the list as sorted, it doesn't affect the underlying ordering. When this is enabled, you can't manually sort the items

Exporter List
Exportez cette liste vers n'importe quel fournisseur de contenu pris en charge par votre téléphone, par ex. la carte SD, l'e-mail, ou Google Drive si l'application Drive est installée, dans un certain nombre de formats différents. Lors de l'exportation vers JSON, cela créera un fichier avec une liste qui pourra ensuite être importée en utilisant l'option "Importer des listes" sur le premier écran.
