Ouvrez une liste en touchant son nom.

Un appui long sur le nom de la liste fera apparaître un menu contextuel qui vous permet de:

  • Renommer
  • Effacer
  • Créer une copie

Créer une copie fera une copie identique de la liste. Vous pouvez l'utiliser pour créer une liste de modèles avec, par exemple, les produits d'épicerie de base que vous achetez toujours.

Sauf si le tri alphabétique est activé, une poignée sur chaque liste vous permet de déplacer la liste vers une autre position dans la liste des listes. Touchez et maintenez pour faire glisser la liste.

Actions

Sauver
Cette action apparaîtra si la liste doit être enregistrée car une sauvegarde précédente a échoué.
Tri actif / inactif
Activez le tri alphabétique. Cela montre uniquement les listes triées, cela n'affecte pas l'ordre sous-jacent. Lorsque cette option est activée, vous ne pouvez pas trier manuellement les listes.
Nouvelle liste
Créer liste nouvelle
Importer liste
Importez des listes. Les listes peuvent provenir de n'importe quel fournisseur de contenu pris en charge par votre appareil, par ex. la carte SD ou Google Drive si l'application Drive est installée. Deux formats sont pris en charge, JSON comme précédemment exporté à partir de Lister, ou le format CSV (valeur séparée par des virgules) avec des colonnes:
Liste nom Article complet ("F" or "T")
e.g.
Achats Pain F
Achats Lait F
Wishes Paix mondiale F
Wishes Maserati F
Le format CSV peut être exporté à partir de feuilles de calcul.
Préférences
