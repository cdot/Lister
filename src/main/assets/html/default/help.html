Main Page

The main page contains a list of all the lists you have created.

A list is opened by clicking the name of the list.

A long click on the list name will bring up a context menu that lets you
  • Rename List
  • Delete
  • Create Copy
Create copy will make an identical copy of the list. This lets you create template list with e.g. your base groceries that you always buy.

Action Bar

New list
Create new list.

Help
Launch this help view.

Settings
Launch the settings view.

Save Backup
Save a backup of all lists to external storage (SD card).

Import Backup
Import files from a previously saved backup. This will add new lists, so the current lists will not be overwritten.
This may lead to duplicate lists in the main view.

List View

The list view is where you manage the content of your lists.

Edit mode
Toggles list editing mode.

Edit Mode

In edit mode you add, remove, and move around items in the list.

When edit is disabled you can't add or remove items. It is still possible to check items and move items.

Items are checked by clicking the check box.
The check box can be placed on either side of the screen, and optionally the entire row can toggle the check box.

A long click on an item lets you rename it or delete it.

Add item
Add the item in the text box to the list.

Move item
Drag item to new place in list.

Menu

  • Rename List
  • Check All Items
  • Uncheck All Items
  • Delete All Checked Items
  • Undo Delete Item
  • Save Backup
  • Restore Backup
Undo will let you put back any deleted item as long as the list is active.
When you exit the list view the history of the list is lost

Save Backup will write the current list out to the SD card.
Restore Backup will restore the current list to the state it had when the latest Save Backup was performed.

Settings

Dim checked items
Lower contrast on checked items so it is easier to see the items that are left to do.

Strike through checked items
Checked items gets crossed out with a line through them.

Move checked items to bottom
The checked items are shown after the unchecked items. This only effects the view. The list is not really re-ordered.
Note: When this is enabled, moving items may not work as expected.

Left hand operation
Default behavior is that the check box and move button is on the right side. This option moves them to the left side.
This allows both left and right handed people to manage the list items with one hand.

Auto-delete checked items
Delete items automatically when they are checked. It is still possible to do un-delete as usual.

Entire row toggles check box
To make it even easier to check items, this option allows the entire row to toggle the check box.

Add new items at top of list
By default items are added at the bottom of the list.

Show list in front of lock screen
If the screen is locked, this option lets you continue using the list without unlocking it.
Note: You still need to unlock the screen before you can access any other app.

Open latest list when app is started
When the app is started it will automatically launch the last open list from the previous session, if it still exists.

Warning before adding duplicate item to list
When a newly entered item has a similar item already in the list; this shows a popup asking for confirmation before the new item is added to the list.

Dark background
When in a list view, this shows screen with white text on dark background. This saves battery on AMOLED displays.

Enable local dictionary
When this is enabled, entered items are remembered and used as suggestions for new items.

Enable automatic backup
When this is enabled, the list is automatically saved to external storage when you exit the list view.

Text size
Set the text size of the list items. Use the system default or manually set Small, Medium or Large
