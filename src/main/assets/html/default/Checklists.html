Lists

Open a list by touching its name.

A long touch on the list name will bring up a context menu that lets you:

  • Rename List
  • Delete
  • Create Copy

Create copy will make an identical copy of the list. You can use this to create template list with e.g. the basic groceries that you always buy.

Action Bar

New list Create a new list
Help Launch this help view
Settings Change settings
Import List Import a list. The list can come from any content provider that your device supports e.g. the SD card or Google Drive. It must be in JSON format previously exported, or in CSV format where the first column is the name of the item, and the second column a checked value ("false", "0", and "" are read as false. Any other value is read as true).

Settings

Change Store
Lets you switch to using a different store. Any store that is accessible through the Android Content Resolver can be used.
Create New Store
Lets you create a new, empty store.
Grey out checked items
Dim checked items so it is easier to see the items that are left to do
Strike through checked items
Cross out checked items with a line
Move checked items to bottom
Show checked items after the unchecked items. This only effects the view, and moving items may not work as expected
Left hand operation
Move the check box and move button to the left side. This makes it easier to manage the list with either hand
Alphabetical Sort
By default, show new lists sorted in alphabetical order
Auto-delete checked items
Delete items automatically when they are checked. You can undo the delete if necessary
Entire row toggles check box
A touch anywhere on the row will toggle the check box
Add new items at top of list
By default items are added at the bottom of the list, check this to add them to the top instead
Show list in front of lock screen
Lets you continue using the list even when the screen is locked. You will still need to unlock the screen before you can access any other app
Open latest list when app is started
Automatically open the last open list from the previous session on startup
Warning before adding duplicate item to list
When a newly entered item is similar to an item already in the list, show a popup asking for confirmation before the new item is added
Text size
Set the text size of the list items
