Open a list by touching its name.

A long touch on the list name will bring up a context menu that lets you:

  • Rename List
  • Delete
  • Create Copy

Create copy will make an identical copy of the list. You can use this to create template list with e.g. the basic groceries that you always buy.

Unless alphabetic sorting is on, a handle on each list lets you move the list to another position in the list of lists. Touch and hold to drag the list.

Action Bar

Sort on/off Toggle alphabetic sorting. This only shows the lists as sorted, it doesn't affect the underlying ordering. When this is enabled, you can't manually sort the lists.
New list Create a new list
Help Launch this help view
Settings Change general settings
Import List Import a list. The list can come from any content provider that your device supports e.g. the SD card, or Google Drive if the Drive app is installed. It must be in JSON format previously exported, or in CSV format where the first column is the name of the item, and the second column a checked value ("false", "0", and "" are read as false. Any other value is read as true)

General Settings

General settings control how the user interface works on this device.
Grey out checked items
Dim checked items so it is easier to see the items that are left to do
Strike through checked items
Cross out checked items with a line
Entire row toggles check box
A touch anywhere on the row will toggle the check box
Show list in front of lock screen
Lets you continue using the list even when the screen is locked. You will still need to unlock the screen before you can access any other app
Left hand operation
Move the check box and move button to the left side. This makes it easier to manage the list with either hand
Show list in front of lock screen
(Try to) keep the list visible, even when the screen is locked. Doesn't work on all devices.
Text size
Set the text size of the list items
Change Store
Lets you switch to using a different store. Any store that is accessible through the Android Content Resolver can be used.
Create New Store
Lets you create a new, empty store.
