Items are checked by touching the check box (or by a touch anywhere on the row, if enabled). Check boxes can be placed on either side of the display.

A long touch on an item lets you rename it or delete it.

Unless sorting is enabled, a handle on each item lets you move the item to another position in the list. Touch and hold to drag the item.

Action Bar

Sort on/off Toggle alphabetic sorting. This only shows the list as sorted, it doesn't affect the underlying ordering. When this is enabled, you can't manually sort the items
Save This action will appear if the list has to be saved because a previous save failed
Add on/off When adding is enabled you can add a new item by entering some text
Uncheck All Items
Check All Items
Delete All Checked Items
Undo Delete Undos the last delete as long as the list is open. When you exit the list view, the undo history is lost
Rename List
Export List Export list to any content provider that your phone supports e.g. the SD card, or email, or Google Drive if the Drive app is installed, in a number of different formats
List settings Change settings that are specific to this list.
